# 番茄钟 App - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**PomoFocus** (番茄专注)

### 1.2 产品定位
一款设计精美、功能完善的番茄钟应用，帮助用户建立专注习惯，提升工作学习效率。

### 1.3 目标用户
- 需要提升专注力的学生和职场人士
- 追求效率的时间管理爱好者
- 喜欢数据追踪的习惯养成者

---

## 2. 功能需求

### 2.1 核心功能

#### 计时器模块
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 标准番茄钟 | 25分钟专注 + 5分钟短休 + 15分钟长休 | P0 |
| 自定义时长 | 可调整专注/休息时长 | P0 |
| 自动循环 | 完成4个番茄后自动长休 | P0 |
| 暂停/跳过 | 支持暂停计时、跳过当前阶段 | P0 |
| 后台运行 | 切换App后计时继续 | P0 |
| 完成提醒 | 计时结束本地通知 + 音效 | P0 |

#### 白噪音模块
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 多种音效 | 雨声/咖啡厅/森林/海浪/白噪音 | P1 |
| 音量调节 | 独立音量控制 | P1 |
| 混合播放 | 可同时播放2种音效 | P2 |

#### 统计模块
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 今日概览 | 完成番茄数/专注时长/打断次数 | P0 |
| 周视图 | 最近7天柱状图 | P0 |
| 月视图 | 整月热力图 (类似GitHub) | P1 |
| 历史记录 | 所有番茄记录列表 | P1 |
| 数据导出 | 导出JSON/CSV | P2 |

#### 设置模块
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 计时器设置 | 时长/自动开始/提示音选择 | P0 |
| 主题切换 | 浅色/深色/跟随系统 | P0 |
| 多主题色 | 5种主题色可选 | P1 |
| 通知设置 | 权限管理/免打扰时段 | P1 |
| 数据管理 | 备份/恢复/清空 | P1 |

### 2.2 特色功能
- **番茄种植**: 完成番茄积累能量，解锁虚拟植物
- **专注森林模式**: 离开App会导致番茄枯萎 (可选)
- **成就系统**: 连续专注天数、累计时长里程碑

## 3. 技术架构

### 3.1 技术栈
```
前端框架: Next.js 15 (App Router)
样式方案: Tailwind CSS + shadcn/ui
状态管理: Zustand
数据存储: IndexedDB (localForage)
跨平台: Capacitor 6 (iOS/Android)
图表库: Recharts
动画库: Framer Motion
音效: Web Audio API + Howler.js
通知: Capacitor Local Notifications
图标: Lucide React
```

### 3.3 数据模型
```typescript
// 番茄记录
interface Pomodoro {
  id: string;
  startedAt: Date;
  endedAt: Date;
  duration: number;        // 实际专注时长(秒)
  plannedDuration: number; // 计划时长(秒)
  wasCompleted: boolean;   // 是否完成
  wasInterrupted: boolean; // 是否被打断
  tags?: string[];         // 标签
}

// 用户设置
interface Settings {
  focusDuration: number;      // 专注时长(默认25分钟)
  shortBreakDuration: number; // 短休(默认5分钟)
  longBreakDuration: number;  // 长休(默认15分钟)
  longBreakInterval: number;  // 几个番茄后长休(默认4)
  autoStartBreak: boolean;    // 自动开始休息
  autoStartFocus: boolean;    // 自动开始专注
  soundEnabled: boolean;      // 音效开关
  soundVolume: number;        // 音量(0-1)
  selectedSound: string;      // 提示音类型
  theme: 'light' | 'dark' | 'system';
  accentColor: string;        // 主题色
}

// 统计数据
interface Stats {
  totalPomodoros: number;
  totalFocusTime: number;     // 总专注时长(分钟)
  currentStreak: number;      // 当前连续天数
  longestStreak: number;      // 最长连续天数
  dailyGoal: number;          // 每日目标(番茄数)
}
```

## 6. 交互细节

### 6.1 计时器交互
- 点击开始: 圆形进度条开始动画
- 背景色随阶段变化 (专注红/短休青/长休蓝)
- 剩余时间显示在中央，字体等宽
- 完成后轻微震动 + 提示音

### 6.2 白噪音交互
- 底部固定面板，可展开/收起
- 点击音效卡片播放，再次点击停止
- 音量滑块实时调节

### 6.3 手势支持
- 左右滑动切换专注/休息模式
- 下拉刷新统计数据
